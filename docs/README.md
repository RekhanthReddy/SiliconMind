# docs/

Drop your PDF manuals here before indexing them in the app.

Examples:
- Tessent_Shell_UserManual.pdf
- ATPG_Reference_Guide.pdf
- MBIST_UserGuide.pdf
- DFT_Interview_QA.pdf
- IEEE_1149_JTAG.pdf

These are NOT required to run the app — the agent works without them.
Upload them via the sidebar in the app for RAG-grounded answers.
